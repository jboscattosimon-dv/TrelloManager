Política de Privacidade

Trello Manager  ·  Última atualização: abril de 2026

Esta Política de Privacidade descreve como o Trello Manager coleta, usa e protege as informações dos usuários ao utilizar o aplicativo e suas integrações com serviços de terceiros (Google Drive e Trello).

1. Quem somos

O Trello Manager é uma ferramenta interna de gestão de conteúdo que permite criar cards no Trello e organizar pastas no Google Drive de forma automatizada. O acesso ao aplicativo é restrito a usuários autorizados.

2. Dados coletados

Coletamos apenas os dados estritamente necessários para o funcionamento do app:

  • Email e nome — usados para autenticação e identificação do usuário dentro do app.
  • Token de acesso ao Google Drive — armazenado de forma segura para permitir a criação e listagem de pastas no Drive do usuário. Nunca lemos, modificamos ou excluímos arquivos além das pastas criadas pelo próprio app.
  • Configurações de clientes e listas do Trello — armazenadas na conta do usuário para personalizar o uso do app.
  • Histórico de ações — registrado localmente no navegador (localStorage) apenas para exibição no Dashboard. Não é enviado a nenhum servidor externo.

3. Como usamos os dados

  • Autenticar o usuário e proteger o acesso ao app.
  • Criar pastas no Google Drive conforme solicitado pelo usuário.
  • Criar e atualizar cards no Trello conforme solicitado pelo usuário.
  • Salvar preferências e configurações para uso futuro.

Não usamos os dados para fins publicitários, análise de comportamento ou qualquer finalidade além das descritas acima.

4. Acesso ao Google Drive

O aplicativo solicita acesso ao Google Drive do usuário exclusivamente para:

  • Listar pastas existentes para seleção pelo usuário.
  • Criar novas pastas de datas dentro de pastas selecionadas pelo usuário.

O app não acessa, lê, modifica ou compartilha nenhum arquivo ou pasta além daquelas explicitamente selecionadas e criadas pelo próprio usuário dentro do app. O token de acesso é armazenado de forma segura e pode ser revogado a qualquer momento pelo usuário, tanto dentro do app quanto nas configurações da conta Google.

5. Compartilhamento de dados

Não vendemos, alugamos nem compartilhamos dados pessoais com terceiros. Os dados são processados pelos seguintes serviços de infraestrutura:

  • Supabase — banco de dados seguro onde as configurações e tokens são armazenados.
  • Google APIs — usadas exclusivamente para as operações de Drive descritas acima.
  • Trello API — usada exclusivamente para criar e atualizar cards conforme solicitado pelo usuário.

6. Segurança

Os tokens de acesso são armazenados de forma segura no banco de dados e nunca expostos no frontend. A comunicação entre o app e os serviços de terceiros ocorre sempre via HTTPS. O acesso ao aplicativo é protegido por autenticação com senha.

7. Retenção de dados

Os dados são mantidos enquanto o usuário tiver uma conta ativa no app. Ao solicitar a exclusão da conta, todos os dados associados — incluindo tokens de acesso ao Google — são permanentemente removidos.

8. Direitos do usuário

O usuário pode, a qualquer momento:

  • Revogar o acesso ao Google Drive dentro do próprio app (perfil → desconectar).
  • Solicitar a exclusão de seus dados entrando em contato pelo email abaixo.
  • Revogar o acesso ao Google diretamente em myaccount.google.com/permissions.

9. Alterações nesta política

Eventuais atualizações nesta política serão publicadas nesta página com a data de revisão atualizada. O uso continuado do app após alterações implica aceitação da política atualizada.
